# Adult Income Dataset – Exploratory Data Analysis

##  Overview

This project analyzes the Adult Income dataset to understand which factors are associated with earning more than 50K per year.

---

##  Objective

* Identify key factors linked to income
* Understand relationships between features and income
* Detect patterns and anomalies

---

## Data Preparation

* Cleaned column names and categorical values
* Handled missing values using "unknown"
* Converted income to binary (0 / 1)
* Removed irrelevant feature (`fnlwgt`)
* Dropped duplicates

---

##  Key Insights

* **Relationship & marital status** show the strongest association with income
* **Education & occupation** have significant impact
* **Numerical features** show weak relationships
* Income distribution is **imbalanced**
* Some anomalies exist, indicating multiple factors influence income

---

##  Visuals

Core insights are shown in the EDA notebook.
Detailed distributions and outlier analysis are in the visualization notebook.

---

##  Conclusion

Income depends on a combination of social and professional factors rather than a single variable.
